# 📚 Java Collection Framework Examples

A comprehensive repository of Java Collection Framework examples designed to help beginners and students understand core concepts with simple and practical programs.

#  Overview

This project demonstrates the usage of the Java Collection Framework, including key interfaces such as:
- List
- Set
- Map
- Queue
Each example is written in a clear and beginner-friendly way to help you understand how data structures work in Java.

# Topics Covered

# List Interface
- ArrayList
- LinkedList
- Vector (optional)

# Set Interface
- HashSet
- LinkedHashSet
- TreeSet
 Map Interface
- HashMap
- LinkedHashMap
- TreeMap

# Queue Interface
- PriorityQueue
- Deque (ArrayDeque)
